# 04 - 微信小程序框架

## 1. 目标与边界
- 提供个人高频使用入口：雷达订阅、提醒查看、打卡执行。
- 一期不做复杂编辑能力，编辑由管理端完成。

## 2. 目录与分包建议
- 主包：首页、消息中心、我的。
- 分包 `radar`：雷达内容浏览。
- 分包 `habit`：打卡与统计。
- 分包 `settings`：订阅与偏好设置。

## 3. 核心能力
- 微信登录与会话绑定。
- 订阅消息授权与模板触达。
- 本地缓存最近内容与离线兜底展示。

## 4. 接口约束
- 仅调用 `BFF-C`，不直连核心服务。
- 每次请求透传设备信息与 `traceId`。

## 5. MVP 验收标准
- 用户可接收提醒并一键进入相关页面。
- 打卡流程在 3 步内完成。
