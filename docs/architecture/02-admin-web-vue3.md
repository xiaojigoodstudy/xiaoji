# 02 - 管理端 Web（Vue3）

## 1. 目标与边界
- 面向 PC 管理员，完成信息源、规则、提醒、打卡管理。
- 一期不做复杂 BI，提供基础看板即可。

## 2. 技术选型
- Vue3 + TypeScript + Vite
- Pinia 状态管理
- Vue Router
- UI：Element Plus（可替换）

## 3. 页面模块
- 仪表盘：抓取量、命中量、发送成功率、打卡完成率。
- 信息雷达：源管理、抓取任务、内容列表。
- 规则引擎：规则 CRUD、启停、测试命中。
- 消息提醒：模板管理、发送日志。
- 打卡管理：事项模板、用户打卡统计。
- 系统管理：用户、角色、权限、配置、审计。

## 4. 前端分层
- `api`：接口定义与请求封装。
- `store`：会话与跨页面状态。
- `views`：页面。
- `components`：复用组件。
- `hooks`：复用逻辑。

## 5. 权限模型
- 菜单权限 + 按钮权限 + 接口权限。
- 前端控制可见性，后端做最终权限校验。

## 6. MVP 验收标准
- 管理员可完成雷达源新增到规则生效全流程。
- 权限不足用户无法访问受限菜单与接口。
