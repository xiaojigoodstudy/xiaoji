# 07 - BFF 与网关

## 1. 目标与边界
- 统一接入策略，隔离客户端差异。
- BFF 负责编排，不承载核心业务规则。

## 2. 网关职责
- 路由转发
- 鉴权校验
- 限流熔断
- 灰度发布
- 访问日志与审计

## 3. BFF 拆分
- `BFF-C`：小程序/公众号/App 公共接口编排。
- `BFF-Admin`：Web/H5 管理端接口编排。

## 4. 接口规范
- 统一错误码与响应结构。
- 写接口支持 `idempotencyKey`。
- 全链路透传 `traceId` 与用户上下文。

## 5. MVP 验收标准
- 三端均通过网关接入。
- 同一业务在不同端返回结构保持可预测一致。
