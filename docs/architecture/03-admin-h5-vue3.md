# 03 - 管理端 H5（Vue3）

## 1. 目标与边界
- 面向移动场景的轻管理：任务查看、规则开关、异常处理。
- 与 PC 管理端复用业务能力，减少重复开发。

## 2. 技术选型
- Vue3 + TypeScript
- 与 Web 管理端共享 `api` 与业务组件（按需）
- 移动 UI：Vant（可替换）

## 3. 功能范围
- 首页概览：当天抓取、命中、发送失败数。
- 快捷操作：规则开关、任务重试、告警确认。
- 日志查询：最近发送日志与抓取失败详情。

## 4. 适配策略
- 路由独立，业务模块复用。
- 组件采用移动优先布局。
- 控制首屏加载，按路由懒加载。

## 5. MVP 验收标准
- 手机浏览器可完成基础运营操作。
- 与 Web 管理端权限体系一致。
