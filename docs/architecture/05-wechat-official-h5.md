# 05 - 公众号 H5 框架

## 1. 目标与边界
- 作为微信生态补充入口，支持内容查看与轻交互。
- 一期用于消息跳转落地页、简单订阅管理。

## 2. 技术与接入
- H5 前端（Vue3）
- 微信 OAuth2 获取身份
- JS-SDK 提供分享与基础能力

## 3. 场景定义
- 模板消息落地：打开对应雷达详情。
- 订阅偏好：开关类目与提醒频率。
- 快捷打卡：从消息直接完成打卡。

## 4. 安全要求
- OAuth 回调域名白名单。
- 接口签名与 token 校验。
- 关键操作二次确认。

## 5. MVP 验收标准
- 公众号消息可准确跳转到目标功能页。
- 登录态稳定，页面刷新后可恢复。
